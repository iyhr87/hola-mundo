Se demostrará que para todo $n \in \mathbb{N}$ se cumple:
\[
3^n < 3^{n+1}.
\]

\textbf{Demostración por inducción:}

\textbf{Caso base:} Para $n=1$:
\[
3^1 = 3 < 9 = 3^2.
\]

\textbf{Paso inductivo:}  
Supongamos que se cumple para un $n \in \mathbb{N}$:
\[
3^n < 3^{n+1}.
\]

Multiplicando por 3:
\[
3 \cdot 3^n < 3 \cdot 3^{n+1}.
\]

Por lo tanto:
\[
3^{n+1} < 3^{n+2}.
\]

\textbf{Conclusión:}  
Se cumple que $3^n < 3^{n+1}$ para todo $n \in \mathbb{N}$.
