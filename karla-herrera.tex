Se demostrará que para todo $n$ que pertenece a los naturales se cumple:
\[
3^n < 3^{n+1}.
\]

 Para $n=1$:
\[
3^1 = 3 < 9 = 3^2.
\]

Supongamos que se cumple para un $n$ que pertenece a los Naturales:
\[
3^n < 3^{n+1}.
\]

Multiplicando por 3:
\[
3(3^n) < 3(3^{n+1}).
\]

Por lo tanto:
\[
3^{n+1} < 3^{n+2}.
\]

Conclusión
Se cumple que $3^n < 3^{n+1}$ para todo $n$ que pertenece a los naturales.
